== Description ==

** this plugin is no longer being update. Please feel free to adopt me! **


The Popularity Stats plugin is a handy plugin for WordPress which quickly reports the popularity of a website by checking with a number of website sources. The Popularity Stats tool includes details about a websites:

- Google PageRank
- Alexa Rank
- Inbound Links

The information is displayed in the WP Admin area of the website and also includes the ability to display the data as a badge for web users to see.
